# Complaint-Management-app
A comprehensive web application designed to streamline the process of registering, managing, and tracking complaints in real time. This system helps organizations efficiently handle user issues across multiple departments with full transparency, faster response times, and improved workflow management.

🚀 Key Features:
🔹 Complaint Registration
    Users can submit complaints quickly with required details.
    Supports attaching documents or images for better clarity.

🔹 Live Complaint Tracking
    Real-time tracking of complaint progress from submission to resolution.
    Automatic status updates like Pending, In Progress, Resolved, Rejected, etc.

🔹 Department-wise Workflow
    Complaints auto-assigned to relevant departments.
    Each department can update status, add notes, and take actions.
    Transparent tracking across Admin, Department Head, and Staff levels.

🔹 Role-Based Access
    Admin: Manage departments, users, complaints, analytics.
    Department Users: Update and resolve assigned complaints.
    End Users: Submit and track their own complaints.

🔹 Real-time Notifications
    Email/SMS/Push notifications for status updates.
    Alerts for pending or delayed complaints.

🔹 Analytics & Reports
    Dashboard showing open, closed, and escalated complaints.
    Export reports for management review.

🔹 Live Activity Tracking
    Monitor complaint movement across departments.
    Track who updated the complaint and when.

🧰 Tech Stack
    Backend: Python & Flask 
    Frontend: HTML, CSS, JavaScript, Bootstrap
    Database: MongoDB
    Others: AJAX, REST API, Real-time updates, FireBase, Nginx
